What is a Warrant Canary?

Broadly speaking, a warrant canary is a methodology for determining that a company or organization has not been served a secret government request for user information. This is established by regularly publishing a "statement that a service provider has not received legal process that it would be prohibited from saying it had received such as a national security letter.


History of Warrant Canaries:

In 2006 the first commercial instance of a warrant canary was published by cloud storage service provider rsync.net. The CanaryWatch organization was created by a coalition of technology companies to raise awareness in the public of secret government national security requests and associated gag orders.


Who would use a Warrant Canary?

Any organization that stores user data could potentially make use of a warrant canary. The only organizations that would be likely to implement a warrant canary of some type are organizations that value user privacy. In the past, some noteable organizations that have used warrant canaries include: Reddit, Pinterest, Apple, CloudFlare, ProtonMail, and various VPN service companies.
